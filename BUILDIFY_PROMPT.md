# Buildify Prompt: Heritage Language Learning Website

## Project Overview
Create a modern, elegant website for "BeyondWords" - an AI-powered tool designed for heritage speakers and diaspora communities to speak their native language with confidence, emotion, and pride. The website should have a warm, welcoming design that emphasizes connection to cultural heritage.

- Main Background: #f5f1ec (warm cream/beige)

## Interactive Elements
- **Hover Effects**: Subtle lift animations on cards and buttons
- **Form Validation**: Email validation for waitlist signups
- **Success States**: Confirmation messages after form submission
- **Responsive Design**: Mobile-friendly with flexible layouts

## Key Features to Implement
1. **Email Collection**: Waitlist signup forms that save to database
2. **Survey Integration**: Link to Microsoft Forms survey
3. **Contact Functionality**: Email client integration
4. **Responsive Design**: Works on all screen sizes
5. **Smooth Animations**: Subtle hover and transition effects

## Design Philosophy
- **Warm and Welcoming**: Use the cream background and rose accents to create a comforting atmosphere
- **Cultural Connection**: Emphasize heritage and family through imagery and language
- **Modern but Accessible**: Clean design that feels contemporary but not intimidating
- **Trust-Building**: Use testimonials and clear value propositions to build confidence

## Technical Requirements
- **Database Integration**: For email collection (Supabase recommended)
- **Form Handling**: Email validation and submission
- **Responsive Layout**: Mobile-first design approach
- **Performance**: Fast loading with optimized images
- **Accessibility**: Screen reader friendly with proper alt text

## Content Focus
The website should emphasize:
- **Heritage Connection**: Reconnecting with cultural roots
- **Family Bonds**: Communication with elders and family members
- **Confidence Building**: Overcoming speaking anxiety
- **Cultural Pride**: Embracing and celebrating heritage languages
- **Community**: Connecting with other heritage speakers

This website serves as a landing page for an AI-powered language learning tool specifically designed for heritage speakers who understand their language but struggle to speak it confidently.
